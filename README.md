DBThrones
=========

Esse projeto visa a criação de uma ontologia _RDF_ para a série _"Crônicas de Gelo e Fogo"_, a partir de dados extraído  da conhecida [A Wiki of Ice and Fire](http://awoiaf.westeros.org/).
